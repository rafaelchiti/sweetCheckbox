sweetCheckbox

Jquery pugin for wrapping standard html checkboxes and style them with a fancy theme BUT still having the default behavior of a normal Input. Does not require extra css files or png files.

WHAT?

sweetCheckbox provides a beautiful and customizable look and feel for your checkboxes. The main goal is to provide a painless integration with html standard checkboxes.

HOW?

Very simple, follow me throught the examples:

< input type="checkbox" id="check1" >
$('#check1').sweetCheckbox();

< input type="checkbox" id="check1" >
$('#check1').sweetCheckbox({'size': 'medium'});

< input type="checkbox" id="check1" >
$('#check1').sweetCheckbox({'size': 'large'});

< input type="checkbox" id="check1" >
$('#check1').sweetCheckbox({'offText': '0', 'onText': '1'});

< input type="checkbox" id="check1" >
$('#check1').sweetCheckbox({'width': '300', 'height': '100', 'fontSize': '32'});
