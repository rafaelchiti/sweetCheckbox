sweetCheckbox

Jquery plugin for wrapping standard html checkboxes and style them with a nice theme BUT still having the default behavior of a normal Input

Download .zip Download .tar.gz View on GitHub

WHAT?

- Are you looking for a checkbox that looks nice and the same on all the standard browsers?

- Do you want to use the checkbox as you usually do, without an strange plugin API or special methods for checking it or getting the value?

Thats it, sweet checkbox is not more than that, a simple plugin that tries to play well with standard html checkboxes.

HOW?

Default

<input type='checkbox' id='check1'>
$('#check1').sweetCheckbox();

Medium Size

<input type='checkbox' id='check2'>
$('#check2').sweetCheckbox({size: 'medium'});

Large Size

<input type='checkbox' id='check3'>
$('#check3').sweetCheckbox({size: 'large'});

Custom Size

<input type='checkbox' id='check4'>
$('#check4').sweetCheckbox({fontSize: '50px', width: '300', height: '80'});

Custom Text

<input type='checkbox' id='check5'>
$('#check5').sweetCheckbox({onText: '1', offText: '0'});
